Задачата избрав да ја решам во Java Spring Boot. Најпрво при старт на апликацијата се извршува Bean кој чита од 
JSON документот и ги зачувува податоците во H2 база. До GET методот на контролерот се пристува на localhost:8080/reviews.
Овде покрај формата за филтрирање се прикажуваат и сите записи од базата. За да не се ореоптоварува табелата, прикажав само дел од
податоците т.е оние кои ми беа потребни за тестирање на филтрите. Во JPA репозиториумот за reviews ги дефинирав функциите кои ми ги даваат 
потребните податоци преку SQL Query. Во сервисниот слој, ги користам овие функции и ги враќам листи со reviews сортирани и филтрирани според
барањата кои ги внел корисникот.
